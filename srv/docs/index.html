Features

  • 5 Application Slots - Deploy up to 5 different Node.js apps
  • PostgreSQL Database - Full database support with persistence
  • Environment Variables - Configure per-application settings
  • GitHub Integration - Webhook-based deployments
  • Process Management - PM2 for reliable application runtime
  • Web Interface - Easy-to-use admin panel

Default Environment Variables

Each deployed application automatically receives:

  • PORT - Application port (3001-3005)
  • SLOT_NAME - Slot identifier (a-e)
  • DATABASE_URL - PostgreSQL connection string
  • POSTGRES_HOST, POSTGRES_PORT, etc. - Database connection details
